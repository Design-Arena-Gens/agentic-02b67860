# منصة المفتش

منصة ويب تفاعلية مبنية بتقنية Next.js لإدارة بيانات المفتشية، التقارير، المتابعات، جداول الإرسال، المراسلات، وجدولة المواعيد مع تنبيهات ذكية.

## المتطلبات

- Node.js 18 أو أحدث
- npm

## التشغيل المحلي

```bash
npm install
npm run dev
```

تُفتح الواجهة على `http://localhost:3000`.

## البناء والإطلاق الإنتاجي

```bash
npm run build
npm start
```

## المزايا الرئيسية

- شاشة دخول سريعة لضبط الوصول للوحة التحكم
- قاعدة بيانات مصغرة لمديري المؤسسات وابتدائيات المديرية
- إدارة تقارير المفتشية مع ملخصات مركزية
- متابعة تقدم خطط دعم المديرين بمؤشرات مرئية
- جداول إرسال للمهام التراسلية مع حالة المتابعة
- حاسبة مردودية تقديرية مبسطة لدعم اتخاذ القرار
- واجهة لحجز رسائل Gmail مع جدول للمراسلات المجدولة
- نظام تسجيل مواعيد مع إبراز المواعيد القريبة خلال 3 أيام
- تخزين محلي للبيانات لضمان استمرارية المعلومات على جهاز المستخدم
